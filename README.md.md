# 📦 Inventory Management System (Work in Progress)
A SQL-driven inventory analytics and management system integrating MySQL database design with a Python-based frontend interface.

## 📌 Project Overview
This project aims to build a **database-driven Inventory Management System** using **MySQL**, **SQL**, and **Python**.  
The system simulates real-world inventory operations such as product tracking, supplier management, stock monitoring, and business analytics.

The goal of the project is to demonstrate practical data analyst and database skills by integrating SQL-based business logic with a Python frontend application.

## 🗄️ Database Overview

The system contains the following core tables:

- **products** – inventory items and stock details  
- **suppliers** – supplier information  
- **shipments** – received inventory records  
- **stock_entries** – stock movement tracking  
- **reorders** – reorder management  

Relationships are implemented using foreign keys to maintain data integrity.

---

## 📊 Business Analytics Implemented

The project currently includes SQL-based business metrics such as:

- Total suppliers count
- Total products available
- Category distribution analysis
- Sales value calculation (last 3 months)
- Restock value tracking
- Low inventory detection without pending reorder

These queries simulate real business reporting scenarios.
